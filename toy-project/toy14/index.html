💙Show and Hide Tab Content Pure CSS💙

주요 Css속성: input, label, 인접선택자, 형제선택자, 속성선택자 :checked, overflow, posotion
☑️오른쪽 ui 탭 버튼 클릭

target 가상클래스를 활용한 순수 css 탭메뉴. 버튼과 content의 id값을 연결하여 공통클래스인 tab클래스에 target속성 부여해서 opacity: 1; visibility: visible;로 클릭하면 나타나는 탭메뉴 설정. 기본값은 opacity: 0; visibility: hidden;으로 보이지 않다가 나타나는 형식. active를 활용해 탭버튼의 색상을 화이트로 변경하는 addClass/removeClass를 적용하였음.
