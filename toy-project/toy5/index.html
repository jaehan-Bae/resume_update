  • How to make this

    코드 전체를 감싸는 부모요소 gallery에 display: flex; 설정 후 자식요소에 flex:1 씩 가로배치. nth-child로 백그라운드 이미지 설정. gallery hover하면 flex:1, li에 hover하면 flex:3, filter:grayscale(1)로 선택한 이미지가 커지며 회색조로 변함. content(설명 박스)는 초기 bottom:-320px으로 밑으로 숨겨두었다가, hover하면 bottom:0px; 설정으로 본래의 자리를 차지하는 이미지 설명 아코디언 완성.
